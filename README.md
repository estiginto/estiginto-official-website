# ESTIGINTO Website

Vite + React static website for the ESTIGINTO official site.

## Requirements

- Node.js 20 or newer
- npm

## Install

```bash
npm ci
```

## Run Locally

```bash
npm run dev
```

Vite prints a local URL, usually:

```text
http://localhost:5173/
```

## Debug

Use the browser DevTools first:

- Console: React/runtime errors
- Network: missing images or failed asset paths
- Elements: layout and CSS inspection
- Lighthouse: performance and SEO checks

Useful commands:

```bash
npm run build
```

Builds the production site into `dist/`. Run this before pushing if you changed React code, assets, or build config.

```bash
npm run preview
```

Serves the built `dist/` output locally. Use this to test the same shape Vercel will deploy.

## Project Structure

```text
index.html              Vite HTML entry and SEO metadata
src/main.jsx            React bootstrap
src/App.jsx             Website content and React components
src/App.css             Website styling
img/                    Images used by the site
scripts/copy-static.mjs Copies public image assets into dist after build
vercel.json             Vercel build/output/cache config
```

## Deploy To Vercel

Vercel should use:

```text
Build Command: npm run build
Output Directory: dist
```

These are already configured in `vercel.json`.

## Notes

- Do not commit `node_modules/` or `dist/`.
- `dist/` is generated by `npm run build`.
- The old static HTML pages are no longer part of the site; navigation is handled inside the React homepage.
